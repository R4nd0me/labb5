HTML Element Node Example: input type text

This HTML document outlines a simple HTML and JavaScript example for utilizing the input element node of type input.

To examine the JS source code, please have a look into the 'scripts.js' file. Enter some text into the text fields below, and submit them by clicking on the respective button. Please open the 'inspector' and 'console' in your browser's developer tools to follow the results of executing the scripts.

  • Firefox: From the menu bar, select Tools → Browser Tools → Web Developer Tools → Console.
  • Safari: From the menu bar, select Develop → Show Web Inspector → Console.
  • Chrome: From the menu bar, select View → Developer → JavaScript Console.
  • Edge: From the menu bar, select Settings and more → More tools → Developer tools.
